





:class:`PMDBExportOptions`
==========================


.. py:class:: ansys.geometry.core.misc.options.PMDBExportOptions

   
   Provides options for PMDB export.


   :Parameters:

       **parameter_prefixes** : :class:`python:str`, default: ""
           Prefixes used to filter parameters for export.

       **cad_attribute_prefixes** : :class:`python:str`, default: ""
           Prefixes used to filter CAD attributes for export.

       **named_selection_prefixes** : :class:`python:str`, default: ""
           Prefixes used to filter named selections for export.

       **analysis_type** : :obj:`AnalysisType`, default: :obj:`AnalysisType.THREE_D`
           The analysis type (2D or 3D).

       **mixed_part_export_type** : :obj:`PMDBMixedPartExportType`, default: :obj:`PMDBMixedPartExportType.NONE`
           The type of mixed parts to export.

       **attach_flattened_assembly** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to attach the assembly in a flattened structure.

       **use_cad_mass_properties** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to use CAD mass properties.

       **plane_prefixes** : :class:`python:str`, default: ""
           Prefixes used to filter planes for export.

       **coordinate_system_prefixes** : :class:`python:str`, default: ""
           Prefixes used to filter coordinate systems for export.

       **advanced_geom_processing** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to enable advanced geometry processing.

       **angular_deviation** : :class:`python:float`, default: 0.0
           Angular deviation for faceting (in degrees).

       **attach_weight_class** : :obj:`PMDBAttachWeightClass`, default: :obj:`PMDBAttachWeightClass.HEAVYWEIGHT`
           The weight class for the attachment.

       **cad_associativity** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to enable CAD associativity.

       **cad_attribute_transfer** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to transfer CAD attributes.

       **do_smart_update** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to perform a smart update.

       **geometry_deviation** : :class:`python:float`, default: 0.0
           Geometry deviation for faceting (in meters).

       **process_coordinate_sys** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to process coordinate systems.

       **process_planes** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to process planes.

       **import_using_instances** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to import using instances.

       **process_work_points** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to process work points.

       **is_selective_update** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to perform a selective update.

       **material_properties** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to include material properties.

       **granta_material_properties** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to include Granta material properties.

       **max_facet_size** : :class:`python:float`, default: 0.0
           Maximum facet size (in meters). Zero means no limit.

       **named_selection** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to export named selections.

       **parameter_processing_type** : :obj:`PMDBImportParameterType`, default: :obj:`PMDBImportParameterType.NONE`
           The type of parameters to process.

       **plug_in_facet_quality** : :obj:`PMDBPlugInFacetQuality`, default: :obj:`PMDBPlugInFacetQuality.NONE`
           The facet quality setting for plug-in readers.

       **process_enclosure_and_symmetry** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to process enclosure and symmetry.

       **reader_save_part** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether the reader should save the part.

       **target_application** : :obj:`PMDBTargetApplication`, default: :obj:`PMDBTargetApplication.PARTMGR`
           The target application for the exported PMDB.

       **temp_directory** : :class:`python:str`, default: ""
           Temporary directory path used during export.

       **process_physics_definition** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to process physics definitions.

       **process_solid_bodies** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to process solid bodies.

       **process_surface_bodies** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to process surface bodies.

       **process_line_bodies** : :ref:`bool <python:bltin-boolean-values>`, default: :data:`python:False`
           Whether to process line bodies.
