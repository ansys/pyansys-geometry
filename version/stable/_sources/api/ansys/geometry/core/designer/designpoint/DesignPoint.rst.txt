   Provides for creating design points in components.


   :Parameters:

       **id** : :class:`python:str`
           Server-defined ID for the design points.

       **name** : :class:`python:str`
           User-defined label for the design points.

       **points** : :obj:`Point3D`
           3D point constituting the design points.
