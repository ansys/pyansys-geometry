ansys.geometry.core.connection.defaults#

Provides default connection parameters.

Module Contents#

ansys.geometry.core.connection.defaults.DEFAULT_HOST#

Default HOST name used.

By default, it searches for the environment variable ANSRV_GEO_HOST, and if it does not exist, it falls back to 127.0.0.1.

ansys.geometry.core.connection.defaults.DEFAULT_PORT: int#

Default HOST port used.

By default, it searches for the environment variable ANSRV_GEO_PORT, and if it does not exist, it falls back to 50051.

ansys.geometry.core.connection.defaults.MAX_MESSAGE_LENGTH#

Default gRPC message length.

By default, it searches for the environment variable PYGEOMETRY_MAX_MESSAGE_LENGTH, and if it does not exist, it falls back to 256Mb.

ansys.geometry.core.connection.defaults.GEOMETRY_SERVICE_DOCKER_IMAGE = 'ghcr.io/ansys/geometry'#

Default Geometry service Docker image location.

Tag is dependent on what OS service is requested.
