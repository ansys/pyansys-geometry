class ansys.geometry.core.sketch.edge.SketchEdge#

Provides for modeling edges forming sketched shapes.

abstract property start: ansys.geometry.core.math.Point2D#

Point that is the start point of the edge.

abstract property end: ansys.geometry.core.math.Point2D#

Point that is the end point of the edge.

abstract property length: pint.Quantity#

Length of the edge.

abstract property visualization_polydata: pyvista.PolyData#

VTK polydata representation for PyVista visualization.

The representation lies in the X/Y plane within the standard global Cartesian coordinate system.

Returns:
pyvista.PolyData

VTK pyvista.Polydata configuration.

plane_change(plane: ansys.geometry.core.math.Plane) → None#

Redefine the plane containing SketchEdge objects.

Parameters:
planePlane

Desired new plane which will contain the sketched edge.

Notes

This implies that their 3D definition may suffer changes. By default, it does nothing. It is required to be implemented in child SketchEdge classes.
