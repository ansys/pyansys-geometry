





:class:`FMDExportOptions`
=========================


.. py:class:: ansys.geometry.core.misc.options.FMDExportOptions(deviation: ansys.geometry.core.misc.measurements.Distance | pint.Quantity | ansys.geometry.core.typing.Real, angle: ansys.geometry.core.misc.measurements.Angle | pint.Quantity | ansys.geometry.core.typing.Real, aspect_ratio: int = -3, max_edge_length: ansys.geometry.core.misc.measurements.Distance | pint.Quantity | ansys.geometry.core.typing.Real = 0.0)

   
   Provides options for FMD export.


   :Parameters:

       **deviation** : :obj:`Distance` | :obj:`Quantity` | :obj:`Real`
           The maximum deviation from the true surface position.
           If a Real is provided, it is assumed to be in the default length unit.
           Must be between 0.00003 m and 0.002 m.

       **angle** : :obj:`Angle` | :obj:`Quantity` | :obj:`Real`
           The maximum deviation from the true surface normal.
           If a Real is provided, it is assumed to be in radians.
           Must be between 0.05 degrees (≈ 8.727e-4 rad) and 30 degrees (≈ 0.5236 rad).

       **aspect_ratio** : :class:`python:int`, default=-3
           The maximum aspect ratio of facets.

       **max_edge_length** : :obj:`Distance` | :obj:`Quantity` | :obj:`Real`, default=0.0
           The maximum facet edge length.
