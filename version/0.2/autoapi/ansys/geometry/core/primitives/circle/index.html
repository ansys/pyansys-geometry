get_parameterization() → ansys.geometry.core.primitives.parameterization.Parameterization#

Return the parametrization of a Circle instance.

The parameter of a circle specifies the clockwise angle around the axis (right hand corkscrew law), with a zero parameter at dir_x and a period of 2*pi.

Returns:
Parameterization

Information about how a circle is parameterized.
